# Urban Spatial Data Dataset

A large-scale urban spatial dataset containing environmental and activity information across different urban zones.

## Dataset Description

This dataset contains urban spatial data with the following features:

- **region_id**: Identifier for the urban region/zone
- **land_use_type**: Type of land use (e.g., Commercial, Residential)
- **temperature**: Temperature level (Low, Medium, High)
- **air_quality_index**: Air quality measurement (Good, Moderate, Poor)
- **noise_level**: Noise level classification (Low, Medium, High)
- **day_of_week**: Day of the week
- **time_of_day**: Time period (Morning, Afternoon, Evening, Night)
- **activity_type**: Type of activity (e.g., Commuting, Recreation)
- **source_type**: Data source type (e.g., GPS, Sensor)
- **traffic_density**: Traffic density level (Low, Medium, High)

## File Information

- **Filename**: `urban_spatial_data_large.csv`
- **Size**: >200MB
- **Format**: CSV (Comma-separated values)

## Usage

This dataset can be used for:
- Urban planning and analysis
- Environmental monitoring studies
- Traffic pattern analysis
- Air quality research
- Noise pollution assessment
